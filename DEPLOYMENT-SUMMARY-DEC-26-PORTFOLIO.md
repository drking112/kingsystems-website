## üîç Troubleshooting

### **If images don't show:**
1. **Hard refresh browser:** Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
2. **Check cache bypass URL** - use `?v=randomnumber` at end
3. **Open incognito window** to bypass all cache
4. **Verify image paths in GitHub:**
   - `images/Portfolio/USAF Academy/Falcon Stadium.png`
   - `images/Portfolio/USAF Academy/Picture2.png`
   - etc.

### **If old content shows:**
1. **Wait 5 minutes** - Cloudflare deployment may still be processing
2. **Check Cloudflare Pages dashboard** - ensure deployment shows "Success"
3. **Clear browser cache completely**
4. **Try different browser**

### **If case study photos are broken:**
- Verify the folder name in GitHub is exactly: `images/Portfolio/USAF Academy/`
- Note: Capital "P" in Portfolio, space between "USAF" and "Academy"
- File extensions are case-sensitive: `.png` vs `.PNG` vs `.jpg` vs `.JPG`
