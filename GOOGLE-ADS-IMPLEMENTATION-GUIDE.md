Quality Score determines:

- Cost per click

**1. Expected CTR (40% weight)**
- Write compelling ad copy
- Use keywords in headlines
- Test multiple ad variations
- Use all ad extensions

**2. Landing Page Experience (40% weight)**
- Fast load time (we're good here - 90+ PageSpeed âœ…)

- Clear CTA (can improve)

**3. Ad Relevance (20% weight)**

- Match user search intent
- Tight keyword groups (8-10 per ad group âœ…)

### Target Quality Scores:
- 7+ = Good
