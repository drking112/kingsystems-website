## Logo Usage Guidelines

### Where These Logos Appear

1. **Homepage - Awards Section**
   - Location: Below CE Pro 100 award
   - Display: 4-column grid with white rounded backgrounds
   - Purpose: Show industry affiliations alongside award

2. **About Page - Professional Section**
   - Location: Below award showcase
   - Display: 4-column grid with descriptions
   - Purpose: Detail association memberships

3. **Footer (Site-wide)**
   - Location: Above CE Pro 100 award badge
   - Display: 4-column grid with white backgrounds
   - Purpose: Site-wide credibility reinforcement

### Display Specifications

**Container Styling**:
- White background: `bg-white`
- Rounded corners: `rounded-lg`
- Padding: `p-4` to `p-6`
- Max height: 80-128px
- Centered alignment

**Logo Sizing**:
- Max height: `max-h-14` to `max-h-20` (56-80px)
- Width: Auto (maintains aspect ratio)
- Object fit: Contain

**Responsive Behavior**:
- Mobile: 2 columns
- Desktop: 4 columns
- Gap: 1.5-2rem between items

### Design Considerations

1. **Consistency**: All logos displayed with equal visual weight
2. **Spacing**: Adequate padding prevents crowding
3. **Hover Effects**: Subtle shadow transitions for interactivity
4. **Accessibility**: Alt text describes each association fully

## Troubleshooting

### Logo Not Displaying
- ✅ Check filename spelling (case-sensitive)
- ✅ Verify file is in correct folder
- ✅ Confirm file format is PNG or JPG
- ✅ Check file permissions

### Logo Looks Distorted
- ✅ Use original aspect ratio
- ✅ Don't set both width AND height
- ✅ Use `object-fit: contain`
- ✅ Provide adequate container space

### Logo Too Small/Large
- ✅ Adjust `max-h-14` to `max-h-20` class
- ✅ Ensure source file is high-resolution
- ✅ Check container padding
- ✅ Verify responsive classes
